# Minecraft: Pocket Community Edition

## About the fork
This is a fork of the Minecraft: Pocket Edition v0.6.1 source code leak that aims to fix a few bugs and clean up the build system.
## About the leak
An anonymous 4chan user [leaked](https://desuarchive.org/g/thread/108264778/#q108266054) the source code of Minecraft: Pocket Edition v0.6.1 on March 1st, 2026

[Here](https://gofile.io/d/ICNxR7) is the original upload of the leak and [here](https://archive.org/download/minecraft-legacy-console-edition-source-code/mc-cpp-snapshot.zip) is an Internet Archive mirror.

The folder structure makes me assume that the leaker downloaded the `master` branch of a repo named `minecraftcpp` via GitHub's webui. Unfortunately, this also means it has no Git history.
